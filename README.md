# GameVault 🎮

مكتبة الألعاب العربية المتطورة

## هيكل المشروع

```
gamevault/
├── public/               ← الموقع العام
│   ├── index.html        ← الصفحة الرئيسية
│   ├── submit.html       ← صفحة إضافة لعبة من المستخدم
│   ├── css/style.css     ← التصميم
│   └── js/app.js         ← المنطق البرمجي
├── admin/                ← لوحة التحكم
│   ├── index.html        ← لوحة الإدارة
│   ├── css/admin.css     ← تصميم الإدارة
│   └── js/admin.js       ← منطق الإدارة
└── data/
    └── games.json        ← قاعدة بيانات الألعاب (733 لعبة)
```

## المميزات

### الموقع العام
- 🎮 **733+ لعبة** مع بيانات كاملة
- 🔍 **بحث فوري** مع اقتراحات ذكية
- 🏷️ **فلاتر ديناميكية** حسب التصنيف
- 📄 **Pagination** - 24 لعبة لكل صفحة
- 🌐 **ثنائي اللغة** عربي/إنجليزي
- 🌙 **وضع داكن وفاتح**
- 📱 **متجاوب** مع الجوال
- 🔗 **مشاركة الألعاب** عبر رابط
- 🛡️ **كشف مانع الإعلانات** مع رسالة ودية

### صفحة إضافة لعبة (submit.html)
- 📝 نموذج متكامل باللغتين
- 🖼️ **رفع صور من الجهاز** بالسحب والإفلات
- ⭐ تقييم بنجوم تفاعلية
- 🤖 **CAPTCHA** لمنع الإرسال التلقائي
- ✅ التحقق من الحقول مع رسائل خطأ
- 📊 شريط تقدم لملء النموذج

### لوحة التحكم (admin/)
- 🔐 **حماية بكلمة مرور** مع session
- 📊 **لوحة إحصائيات** شاملة
- 📥 **مراجعة طلبات المستخدمين** - قبول أو رفض
- ➕ إضافة وتعديل وحذف الألعاب
- 🔍 بحث وفلترة في جدول الألعاب
- ⬇️ تصدير games.json
- 🔗 **ربط GitHub** للنشر التلقائي
- 🔒 تغيير كلمة المرور

## البدء

1. افتح `public/index.html` في المتصفح
2. افتح `admin/index.html` للوحة التحكم
3. كلمة المرور الافتراضية: `admin123`

## GitHub Pages

1. ارفع المشروع على GitHub
2. فعّل GitHub Pages من إعدادات الـ repo
3. ادخل على لوحة التحكم > إعدادات > GitHub
4. أدخل توكن GitHub وبيانات الـ repo
5. كل تغيير ستحفظه سيُنشر تلقائياً

## أمان

- كلمة مرور الأدمن مشفرة في localStorage
- CAPTCHA رياضي لمنع السبام
- تنظيف HTML لمنع XSS
- Session تنتهي بعد 8 ساعات
